Sobre Nós

Pure Wood trata-se de uma aplicação web que simula uma loja de venda de madeira e produtos derivados de madeira.

utilizador poderá escolher entre comprar madeira pura de várias qualidades ou poderá escolher mobílias produzidos do mesmo material.

Para além disso o próprio utilizador poderá "criar" a sua própria mobília através de um sistema de "várias peças" em que o próprio poderá escolher vários tipos de madeiras diferentes para criar uma mobília com o tamanho adequado que o utilizador prefira.

Quaisqueres dados fornecidos pelo utilizador não seram utilizados para fins maliciosos e simplesmente seram utilizados para o proposito da aplicação web.
